# Valentine Surprise Site

Simple mobile-first one-page site with:

1. Question screen
2. Tap-to-swap 3x3 puzzle gate
3. Final memory gallery + note

## Replace media

1. Replace `assets/us-puzzle.svg` with your real puzzle photo (recommended square image).
2. Replace `assets/photo-1.svg`, `assets/photo-2.svg`, and `assets/photo-3.svg` with your gallery photos.
3. If you switch file names or extensions, update paths in `index.html`.

## Edit text

1. Main question text is in `index.html`.
2. Final note text is in `index.html`.

## Run locally

Use any static server, for example:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
